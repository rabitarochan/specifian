# {title}

API for retrieving, creating, and updating user information.

Status: <StatusBadge status="published" />

<DataView data={data.endpoint} />

## Request Example

```bash
GET /api/users?page=1&limit=20&role=member
Authorization: Bearer <token>
```

## Response Example

| Status Code | Description |
|---|---|
| 200 | Successfully retrieved |
| 400 | Invalid parameters |
| 401 | Authentication required |
| 500 | Server error |

### 200 OK

```json
{
  "data": [
    {
      "id": 1,
      "email": "user@example.com",
      "name": "User Name",
      "role": "member",
      "created_at": "2026-01-01T12:00:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 100
  }
}
```

### 401 Unauthorized

```json
{
  "error": "Invalid or missing authentication token"
}
```

## Design Notes

- Bearer token authentication is used.
- Pagination is required to prevent fetching large amounts of data at once.
- Filtering by the `role` parameter is supported.
