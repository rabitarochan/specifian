# {title}

ユーザー情報の取得・作成・更新を行う API。

<DataView data={data.endpoint} />

## リクエスト例

```bash
GET /api/users?page=1&limit=20&role=member
Authorization: Bearer <token>
```

## レスポンス例

| ステータスコード | 説明 |
|---|---|
| 200 | 取得成功 |
| 400 | パラメータが不正 |
| 401 | 認証が必要 |
| 500 | サーバーエラー |

### 200 OK

```json
{
  "data": [
    {
      "id": 1,
      "email": "user@example.com",
      "name": "User Name",
      "role": "member",
      "created_at": "2026-01-01T12:00:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 100
  }
}
```

### 401 Unauthorized

```json
{
  "error": "Invalid or missing authentication token"
}
```

## 設計メモ

- 認証には Bearer トークンを使用します
- ページネーション必須で、大量のデータ取得を防いでいます
- role パラメータでのフィルタリングをサポートしています
