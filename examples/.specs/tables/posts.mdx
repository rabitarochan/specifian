# {title}

Table for managing content posted by users. Supports blog articles, news posts, and similar content.

<TableDefinition data={data.table} />

## Design Notes

- `user_id` is a foreign key referencing the [[tables:users]] table.
- The default value for `status` is `'draft'`; no automatic publishing is performed.
- `published_at` can also be used to schedule a future publication date.
- If soft delete is required, set `status` to `'archived'` instead of physically deleting the row.
